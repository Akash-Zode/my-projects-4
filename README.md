# Employee Payroll Management System (Java)

## Overview

The Employee Payroll Management System is a Java-based console application designed to manage employee details and automate salary calculations. It allows users to add employees, view individual payroll details, and display all employee records in an efficient manner.

This project demonstrates core Java concepts such as **Object-Oriented Programming (OOP)**, **ArrayList**, and **user input handling** using Scanner.


## Features

* Add new employee details
* View specific employee payroll using ID
* View all employees
* Automatic salary calculation (HRA, DA, deductions, net salary)
* Simple and user-friendly console interface

---

## Salary Calculation Logic

* HRA = 20% of Basic Salary
* DA = 10% of Basic Salary
* Gross Salary = Basic + HRA + DA
* Deductions = 8% of Gross Salary
* Net Salary = Gross - Deductions
